Treasure Adventure Game

Pirate themed metroidvania

Developer: Robit

Genre: Action, Adventure, Platformer

[Description WIP] Treasure Adventure Game is a huge open ended adventure game that takes place across several islands and dungeons. The goal is to find 12 different treasures scattered throughout the game world using a variety of abilities and items. There is a day and night cycle and NPC's adapt to time of day. The game is fairly long compared to other freeware and can take anywhere between 15-20 hours to complete. The game is available to download off the author's website linked below.
